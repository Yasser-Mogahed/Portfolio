# Yasser Mogahed — Portfolio Website

> Neo-Cyberpunk Spider-Man themed personal portfolio of **Yasser Ahmed Mogahed** — Data Scientist & BI Engineer.

---

## 📁 Project Structure

```
portfolio/
├── index.html      # Main HTML structure
├── css/
│   └── style.css   # All styles, animations & responsive layout
└── js/
    └── script.js   # Interactions: loader, cursor, typed text, neural canvas, scroll
```

> **Note:** The `css/` and `js/` folder references are already linked in `index.html`.  
> Just move `style.css` into a `css/` subfolder and `script.js` into a `js/` subfolder — or update the paths in `index.html` to keep them flat.

---

## 🚀 Features

- Loading screen with animated progress bar
- Custom neo-cyberpunk cursor system
- Animated neural canvas background (WebGL-like particle network)
- Spider-Sense radial lines + hexagonal photo frame
- Typed role animation (Data Scientist · BI Engineer · ML Engineer · Team Leader)
- Scroll-triggered reveal animations
- Certifications drag-to-scroll carousel
- Parallax hero background text
- Active nav link tracking on scroll
- Fully responsive (mobile, tablet, desktop)

---

## 🛠 Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JavaScript** — zero frameworks
- Google Fonts: Orbitron, Bebas Neue, Rajdhani, Share Tech Mono, Exo 2
- Canvas API for neural network animation

---

## 📌 Usage

Simply open `index.html` in a browser — no build step required.

```bash
# Clone and open
git clone https://github.com/Yasser-Mogahed/<repo-name>.git
cd <repo-name>
open index.html
```
